# 🎲 Dice Rolling Simulator - Flask Web App

A simple Dice Rolling Simulator built using **Python** and **Flask**.  
This web application runs on **localhost** and simulates a real dice roll with animation.

---

## 🚀 Features

- 🎲 Animated dice rolling effect
- 🔢 Displays final rolled number clearly
- 🌐 Runs in web browser (localhost)
- ⚡ Uses JavaScript + Flask backend
- 🔁 Roll unlimited times

---

## 🛠 Technologies Used

- Python 3.x
- Flask
- HTML
- CSS
- JavaScript (Fetch API)

## 🎯 How It Works

1. Click **Roll Dice** button.
2. Dice animation starts (rapid number changes + shake effect).
3. After 1 second, animation stops.
4. Flask backend generates a random number (1–6).
5. Final result is displayed clearly on screen.

---

## 🧠 Backend Logic

- `/` → Loads main page.
- `/roll` → Returns a random number (1–6) in JSON format.
- JavaScript fetches the number and updates the UI.
